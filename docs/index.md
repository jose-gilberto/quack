# Welcome to quack 🦆

**quack** (Quantification Kit) is a library with basic tools for quantification, or prevalence estimation.

## Instalação

```bash
pip install git+[https://github.com/jose-gilberto/quack.git](https://github.com/jose-gilberto/quack.git)
```
